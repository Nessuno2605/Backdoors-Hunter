# Backdoors-Hunter

Tool di analisi connessioni e processi per Windows, progettato per individuare possibili backdoor o attività sospette.

## Funzionalità
- Analisi delle connessioni attive
- Verifica dei processi in esecuzione
- Controllo porte aperte
- Database locale per registrare risultati

## Requisiti
- Windows 11
- Python 3.x
