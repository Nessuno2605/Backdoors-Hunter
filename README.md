# Backdoors-Hunter

Backdoors-Hunter è un tool di analisi per sistemi Windows progettato per individuare possibili backdoor, connessioni sospette e processi anomali.  
Pensato per tecnici, analisti e appassionati di cybersecurity, permette di effettuare controlli rapidi e mirati sulla macchina.

---

## 🔍 Funzionalità principali

- **Analisi delle connessioni attive**  
  Utilizza `netstat` per identificare porte aperte, connessioni remote e processi associati.

- **Verifica dei processi in esecuzione**  
  Individua processi potenzialmente malevoli o non riconosciuti.

- **Controllo porte aperte e servizi attivi**

- **Database locale SQLite**  
  Registra risultati, cronologia e analisi precedenti.

- **Compatibilità Windows 10/11**

---

## 🛠️ Tecnologie utilizzate

- Python 3.x  
- SQLite3  
- Moduli standard: `subprocess`, `platform`, `datetime`, `hashlib`, ecc.
